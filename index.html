Les bases du CSS

Les selecteurs

  • les sélecteurs de type : permet de cibler les éléments qui correspondent au nom indiqué
    ex : input permettra de cibler n'importe quel élément <input>
  • les sélecteurs de classe : permet de cibler les éléments en fonction de la valeur de leur attribut class
    ex : .index permettra de cibler n'importe quel élément qui possède la classe index (class="index")
  • les selecteurs d'identifiants : permet de cibler un élément d'un document en fonction de la valeur de son attribut id
    ex : #toc permettra de cibler l'élément qui possède l'identifiant toc (id="toc")
  • Le sélecteur universel : permet de cibler tous les élements
    ex : * permettra de cibler tous les éléments du document

Les propriétés

texte

  • color : permet de changer la couleur du texte
  • font-size : permet de gérer la taille de la police
  • font family : permet de gérer la police d'écriture
  • font-weight : permet de choisir l'épaisseur la police
  • font-style : permet d'écrire en italique
  • line-height : permet de choisir la hauteur minimale des lignes
  • text-decoration : permet de "décorer le texte" (souligner par exemple)

Les bordures

  • border-color : permet de choisir la couleur des bordures
  • border-size : permet de choisir l'épaisseur de la bordure
  • border-style : permet de choisir le type de bordure
  • border-radius : permet de définir les coins arondis des bordures

Box-model

  • margin : permet de définir la taille des marges sur les quatre côtés de l'élément
  • padding : permet de définir les différents écarts de remplissage d'un élément, sur l'ensemble de ses côtés

Autres

  • backgroung-color : permet de choisir la couleur de fond
  • backgroung-image : permet de choisi l'image de fond
  • width : permet de déterminer la largeur
  • height : permet de déterminer la hauter
